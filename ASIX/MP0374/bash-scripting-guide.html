🐚 Bash Scripting —
De Zero a Heroi

Bash és l'intèrpret d'ordres per defecte a la majoria de distribucions Linux. Aprendre a escriure scripts de Bash et permet automatitzar tasques repetitives, gestionar fitxers i processos, i crear eines pròpies de línia d'ordres. Aquesta guia t'acompanya des del primer script fins a tècniques avançades.

Primers Passos

Un script de Bash és un fitxer de text amb una llista d'ordres. La primera línia (el shebang) indica al sistema operatiu quin intèrpret ha d'usar per executar-lo.

El primer script

Crea el fitxer

Per convenció els scripts de Bash tenen l'extensió .sh, però no és obligatori.

Escriu el contingut

El shebang #!/bin/bash a la primera línia és essencial.

Fes-lo executable i executa'l

Cal donar permís d'execució amb chmod abans de poder-lo executar.

Bash# Pas 1: Crea el fitxer
nano hola.sh    # o vim, gedit, etc.
hola.sh#!/bin/bash
# Això és un comentari — Bash l'ignora
# Primer script: Hola, món!

echo "Hola, món!"
echo "Avui és: $(date +%A, %d/%m/%Y)"
echo "Ets l'usuari: $USER"
Bash# Pas 3: Permís d'execució i execució
chmod +x hola.sh
./hola.sh
SortidaHola, món!
Avui és: dilluns, 05/03/2026
Ets l'usuari: anna

Variables

Les variables emmagatzemen dades temporalment. A Bash tot és text per defecte — no hi ha tipus de dades com en altres llenguatges. La sintaxi és senzilla però cal respectar les espais (o l'absència d'espais).

Declaració i ús

# Ús: amb $ davant del nom
echo "Hola, $nom!"
echo "Tens ${edat} anys"    # {} per delimitar el nom

# IMPORTANT: espais trenquen l'assignació
# nom = "Anna"   ← ERROR! Bash ho interpreta com una ordre

# Variable de només lectura (constant)
readonly PI=3.14159

# Eliminar una variable
unset nom
Cometes: quan i per què
Bashfrase="Hola món"

# Cometes dobles: expanden variables i subshells

# Cometes simples: text literal, NO expandeixen res
echo 'La frase és: $frase'      # → La frase és: $frase

# Sense cometes: el shell divideix per espais (word splitting)
echo $frase                    # → Hola món  (igual, però perillós amb espais)

# Bona pràctica: SEMPRE posa les variables entre cometes dobles

# rm $fitxer    ← DOLENT: intentaria borrar "el", "meu" i "document.txt"
rm "$fitxer"    # ← CORRECTE

Entrada i Sortida

Bash gestiona tres fluxos estàndard: stdin (entrada), stdout (sortida) i stderr (errors). Dominar la redirecció i els pipes és fonamental per construir scripts potents.

echo i printf
Bash# echo — sortida simple
echo "Hola, món!"
echo -n "Sense salt de línia"    # -n: no afegeix \n al final
echo -e "Línia 1\nLínia 2"       # -e: interpreta seqüències d'escapament

# printf — formatació avançada (com en C)
printf "Nom: %-15s Edat: %3d\n" "Anna" 25
# → Nom: Anna            Edat:  25

printf "Float: %.2f\n" 3.14159
# → Float: 3.14

# Colors amb echo -e (còdis ANSI)
echo -e "\e[32mVerd\e[0m normal \e[31mVermell\e[0m"

# Millor amb variables:
VERD='\e[32m' VERMELL='\e[31m' NC='\e[0m'  # NC = No Color
echo -e "${VERD}Èxit!${NC}"

Bucles

Els bucles permeten repetir blocs de codi. Bash n'ofereix tres tipus principals: for (iteració sobre llistes), while (mentre una condició sigui certa) i until (fins que una condició sigui certa).

for — iteració sobre llistes
Bash# Iteració sobre valors literals
for fruit in poma pera taronja maduixa; do
    echo "Fruit: $fruit"
done

# Iteració sobre un rang numèric
for i in {1..5}; do
    echo "Número: $i"
done

# Rang amb pas
for i in {0..20..5}; do   # 0, 5, 10, 15, 20
    echo $i
done

# Sintaxi estil C
for (( i=0; i<5; i++ )); do
    echo "i = $i"
done

# Iteració sobre fitxers
for fitxer in *.txt; do
    echo "Processant: $fitxer"
done

# Iteració sobre la sortida d'una ordre
for usuari in $(cat /etc/passwd | cut -d: -f1 | head -5); do
    echo "Usuari: $usuari"
done
while i until
Bash# while — executa MENTRE la condició sigui certa
comptador=1
while [[ $comptador -le 5 ]]; do
    echo "Compte: $comptador"
    (( comptador++ ))
done

# until — executa FINS QUE la condició sigui certa (invers de while)
n=1
until [[ $n -gt 5 ]]; do
    echo "n = $n"
    (( n++ ))
done

# Llegir un fitxer línia per línia
while IFS= read -r linia; do
    echo "Línia: $linia"
done < "fitxer.txt"

# Bucle infinit amb break
while true; do
    read -p "Ordre (q=sortir): " entrada
    [[ "$entrada" == "q" ]] && break
    echo "Has dit: $entrada"
done
break i continue
Bash# break — surt del bucle
for i in {1..10}; do
    if [[ $i -eq 5 ]]; then
        break
    fi
    echo $i    # imprimeix 1 2 3 4
done

# continue — salta a la pròxima iteració
for i in {1..6}; do
    if [[ $(( i % 2 )) -eq 0 ]]; then
        continue    # salta els parells
    fi
    echo $i    # imprimeix 1 3 5
done

Funcions

Les funcions agrupen blocs de codi reutilitzables. A Bash, les funcions reben arguments igual que els scripts (amb $1, $2...) i retornen un codi de sortida numèric. Per retornar text, s'usa echo i substitució de comandes.

Definició i crida
Bash# Sintaxi 1 (recomanada)
saluda() {
    local nom=${1:-"Món"}    # local: variable local a la funció
    echo "Hola, $nom!"
}

# Sintaxi 2 (amb paraula clau function)
function adeu() {
    echo "Adeu, $1!"
}

# Crida de la funció
saluda "Anna"    # → Hola, Anna!
saluda           # → Hola, Món!
adeu "Pere"      # → Adeu, Pere!
Retorn de valors
Bash# return → retorna un codi de sortida (0-255)
# 0 = èxit, ≠0 = error
es_parell() {
    if [[ $(( $1 % 2 )) -eq 0 ]]; then
        return 0    # èxit (veritable)
    else
        return 1    # error (fals)
    fi
}

es_parell 4 && echo "4 és parell"
es_parell 3 || echo "3 és imparell"

# Retornar text: echo + substitució de comandes $( )
majuscules() {
    echo "$1" | tr '[:lower:]' '[:upper:]'
}

resultat=$(majuscules "hola món")
echo "$resultat"    # → HOLA MÓN

Arrays

Bash suporta arrays indexats (llistes) i arrays associatius (diccionaris). Útils per gestionar col·leccions de valors sense haver de crear múltiples variables.

Arrays indexats
Bash# Declaració
fruites=("poma" "pera" "taronja" "maduixa")

# Accés per índex (comença en 0)
echo ${fruites[0]}       # → poma
echo ${fruites[2]}       # → taronja
echo ${fruites[-1]}      # → maduixa (últim element)

# Tots els elements
echo ${fruites[@]}       # → poma pera taronja maduixa

# Nombre d'elements
echo ${#fruites[@]}      # → 4

# Afegir elements
fruites+=("raïm")
fruites[10]="kiwi"      # índex no consecutiu

# Eliminar un element
unset fruites[1]

# Iteració
for fruit in "${fruites[@]}"; do
    echo "- $fruit"
done

# Iteració amb índex
for i in "${!fruites[@]}"; do
    echo "[$i] ${fruites[$i]}"
done

# Subarray
echo "${fruites[@]:1:2}"  # des de índex 1, 2 elements
Arrays associatius (Bash 4+)
Bash# Cal declarar-los explícitament amb declare -A
declare -A capitals

capitals["Espanya"]="Madrid"
capitals["França"]="París"
capitals["Alemanya"]="Berlín"
capitals["Catalunya"]="Barcelona"

# Accés per clau
echo ${capitals["Catalunya"]}     # → Barcelona

# Totes les claus
echo "${!capitals[@]}"

# Tots els valors
echo "${capitals[@]}"

# Iteració
for pais in "${!capitals[@]}"; do
    echo "Capital de $pais: ${capitals[$pais]}"
done

Manipulació de Text

El processament de text és el punt fort de Bash. Combina les expansions de cadenes integrades amb les eines Unix clàssiques: grep, sed, awk, cut, tr i sort.

Expansions de cadenes ${}
Bashtext="Hola, Món!"
fitxer="document.backup.txt"

# Longitud
echo ${#text}                    # → 10

# Convertir majúscules/minúscules (Bash 4+)
echo ${text^^}                   # → HOLA, MÓN! (tot majúscules)
echo ${text,,}                   # → hola, món! (tot minúscules)
echo ${text^}                    # → Hola, Món! (primera majúscula)

# Substring: ${variable:posició:longitud}
echo ${text:0:4}                 # → Hola
echo ${text: -4}                 # → Món! (des del final)

# Eliminar prefix (# = mínim, ## = màxim)
echo ${fitxer#*.}               # → backup.txt (elimina fins al primer .)
echo ${fitxer##*.}              # → txt (elimina fins a l'últim .)

# Eliminar sufix (% = mínim, %% = màxim)
echo ${fitxer%.*}               # → document.backup (elimina des de l'últim .)
echo ${fitxer%%.*}              # → document (elimina des del primer .)

# Reemplaçar: ${variable/patró/substitució}
echo ${text/Hola/Adeu}          # → Adeu, Món! (primera ocurrència)
echo ${text//o/0}               # → H0la, M0n! (totes les ocurrències)
grep — cerca de patrons
Bash# Cercar text en fitxers
grep "ERROR" app.log
grep -i "error" app.log          # -i: insensible a majúscules
grep -n "ERROR" app.log          # -n: mostra número de línia
grep -c "ERROR" app.log          # -c: compta coincidències
grep -v "ERROR" app.log          # -v: línes que NO coincideixen
grep -r "TODO" ./src/           # -r: recursiu en directoris

# Expressions regulars exteses
grep -E "^[0-9]{4}-[0-9]{2}" dades.txt
grep -E "(ERROR|WARN)" app.log

# Mostrar context (línies de context)
grep -A 3 -B 1 "ERROR" app.log  # 3 línies despres, 1 abans
sed — edició de flux
Bash# Substitució: s/patró/substitució/flags
sed 's/hola/adeu/' fitxer.txt      # primera ocurrència per línia
sed 's/hola/adeu/g' fitxer.txt     # g = totes les ocurrències
sed 's/hola/adeu/gi' fitxer.txt    # gi = totes + insensible

# Edició al fitxer original (-i)
sed -i 's/versio=1/versio=2/g' config.ini

# Eliminar línies
sed '/^#/d' fitxer.txt             # elimina línies que comencen per #
sed '/^$/d' fitxer.txt             # elimina línies buides

# Imprimir línies específiques
sed -n '5,10p' fitxer.txt          # línies 5 a 10
sed -n '/INICI/,/FINAL/p' fitxer.txt
awk — processament de columnes
Bash# Imprimir columnes específiques (separador per defecte: espai)
awk '{print $1, $3}' fitxer.txt    # columnes 1 i 3
awk -F: '{print $1}' /etc/passwd   # -F: separador de camps

# Condicionals en awk
awk '$3 > 100 {print $1, $3}' dades.txt
awk '/ERROR/ {print NR": "$0}' app.log  # NR = número de línia

# Sumar una columna
awk '{suma += $2} END {print "Total:", suma}' vendes.txt

# Recompte de paraules
awk '{for(i=1;i<=NF;i++) freq[$i]++} END {for(w in freq) print freq[w], w}' \
  text.txt | sort -rn | head -10
cut, sort, uniq, tr, wc
Bash# cut — extreure columnes o caràcters

sort -k2 -n fitxer.txt            # ordena per columna 2

# uniq — elimina/compta duplicats (requereix entrada ordenada)
sort fitxer.txt | uniq           # elimina duplicats
sort fitxer.txt | uniq -c        # compta cada línia única
sort fitxer.txt | uniq -d        # mostra solo duplicats

# tr — transliterar/eliminar caràcters
echo "hola" | tr 'a-z' 'A-Z'       # → HOLA
echo "a,b,c" | tr ',' '\n'         # substitueix coma per salt de línia
echo "abc123" | tr -d '0-9'       # elimina dígits

# wc — comptar línies, paraules, caràcters
wc -l fitxer.txt                  # nombre de línies
wc -w fitxer.txt                  # nombre de paraules
wc -c fitxer.txt                  # nombre de bytes

Gestió d'Errors

Els scripts robustos gestionen els errors de forma explícita. Bash ofereix eines per aturar-se davant d'errors, capturar senyals del sistema i retornar codis de sortida informatius.

Opcions de seguretat: set -euo pipefail
Bash#!/usr/bin/env bash
set -euo pipefail
# -e : atura el script si qualsevol ordre retorna un codi ≠ 0
# -u : error si s'usa una variable no definida
# -o pipefail : el pipe falla si qualsevol part del pipe falla

# Amb -e, a vegades cal permetre que algunes ordres fallin:
count=$(grep -c "patró" fitxer.txt || true)  # || true evita la sortida per grep -c

# O desactivar temporalment:
set +e
resultat=$(ordre_que_pot_fallar)
codi=$?
set -e
Codis de sortida i $?
Bash# Convencions de codis de sortida:
# 0   = èxit
# 1   = error general
# 2   = ús incorrecte de la ordre o argument invàlid
# 126 = permís denegat
# 127 = ordre no trobada

# Comprovar el codi de sortida de l'última ordre
cp fitxer.txt /dest/
if [[ $? -ne 0 ]]; then
    echo "Error copiant el fitxer" >&2
    exit 1
fi

morir() {
    echo -e "\e[31m[ERROR]\e[0m $*" >&2

[[ -f "config.ini" ]] || morir "No es troba config.ini"

ajuda() {
    cat <<EOF

Script de monitoratge del sistema
monitor.sh#!/usr/bin/env bash
set -euo pipefail

# Llindars d'alerta
LLINDAR_CPU=80
LLINDAR_DISC=90
LLINDAR_MEMORIA=85

comprova_cpu() {
    us_cpu=$(top -bn1 | grep "Cpu(s)" | awk '{print $2}' | cut -d. -f1)
    if [[ $us_cpu -gt $LLINDAR_CPU ]]; then
        echo -e "\e[31m[ALERTA] CPU: ${us_cpu}% (llindar: ${LLINDAR_CPU}%)\e[0m"
    else
        echo -e "\e[32m[ OK  ] CPU: ${us_cpu}%\e[0m"
    fi
}

comprova_disc() {
    while IFS= read -r linia; do
        percentatge=$(echo "$linia" | awk '{print $5}' | tr -d '%')
        punt=$(echo "$linia" | awk '{print $6}')
        if [[ $percentatge -gt $LLINDAR_DISC ]]; then
            echo -e "\e[31m[ALERTA] Disc $punt: ${percentatge}% ple\e[0m"
        else
            echo -e "\e[32m[ OK  ] Disc $punt: ${percentatge}%\e[0m"
        fi
    done < <(df -h | grep '^/dev')
}

comprova_memoria() {
    total=$(free | grep Mem | awk '{print $2}')
    usada=$(free | grep Mem | awk '{print $3}')
    percentatge=$(( usada * 100 / total ))
    if [[ $percentatge -gt $LLINDAR_MEMORIA ]]; then
        echo -e "\e[31m[ALERTA] Memòria: ${percentatge}%\e[0m"
    else
        echo -e "\e[32m[ OK  ] Memòria: ${percentatge}%\e[0m"
    fi
}

echo "=== Informe del sistema: $(date) ==="
comprova_cpu
comprova_disc
comprova_memoria

Bàsic Ex. 1 — Calculadora de l'IMC

Escriu un script que demani a l'usuari el seu pes (en kg) i la seva alçada (en metres), calculi l'Índex de Massa Corporal (IMC = pes / alçada²) i mostri el resultat amb la categoria corresponent.

  • IMC < 18.5 → Baix pes
  • 18.5 ≤ IMC < 25 → Pes normal
  • 25 ≤ IMC < 30 → Sobrepès
  • IMC ≥ 30 → Obesitat

Pista: Usa bc per fer el càlcul amb decimals.

Solució#!/usr/bin/env bash
set -euo pipefail

read -p "Pes (kg): " pes
read -p "Alçada (m, ex: 1.75): " alcada

# Càlcul amb bc (escala 2 decimals)
imc=$(echo "scale=2; $pes / ($alcada * $alcada)" | bc)

echo "IMC: $imc"

# Comparació amb bc (retorna 1 si és cert, 0 si és fals)
menor_que() { echo "$1 < $2" | bc -l; }

if [[ $(menor_que "$imc" "18.5") -eq 1 ]]; then
    echo "Categoria: Baix pes"
elif [[ $(menor_que "$imc" "25") -eq 1 ]]; then
    echo "Categoria: Pes normal"
elif [[ $(menor_que "$imc" "30") -eq 1 ]]; then
    echo "Categoria: Sobrepès"
else
    echo "Categoria: Obesitat"
fi

Mitjà Ex. 4 — Organitzador de fitxers per extensió

Escriu un script que organitzi tots els fitxers d'un directori (passat com a argument) en subdirectoris per extensió. Per exemple: foto.jpg va a jpg/foto.jpg, doc.pdf va a pdf/doc.pdf.

  • Ha de ignorar els directoris (només mou fitxers)
  • Si no hi ha extensió, el fitxer va a sense_extensio/
  • Mostra quants fitxers ha mogut al final
Solució#!/usr/bin/env bash
set -euo pipefail

DIR=${1:-.}    # per defecte: directori actual
[[ -d "$DIR" ]] || { echo "Error: '$DIR' no és un directori"; exit 1; }

moguts=0

for fitxer in "$DIR"/*; do
    # Salta si no és un fitxer regular
    [[ -f "$fitxer" ]] || continue

    nom=$(basename "$fitxer")
    ext=${nom##*.}

    # Si nom i ext són iguals, no té extensió
    if [[ "$nom" == "$ext" ]]; then
        ext="sense_extensio"
    fi

    desti="$DIR/$ext"
    mkdir -p "$desti"
    mv "$fitxer" "$desti/$nom"
    (( moguts++ ))
    echo "  $nom → $ext/"
done

echo
echo "Total: $moguts fitxer(s) organitzat(s)"
Avançat Ex. 5 — Script de desplegament senzill

Crea un script de desplegament que:

  • Accepti un argument: l'entorn (dev, staging, prod)
  • Per a prod, demani confirmació explícita a l'usuari
  • Faci una còpia de seguretat del directori de desplegament actual

restaurar() {

        log "ERROR detectat. Restaurant backup..."
        rm -rf "$DIR_DESTI"
        mv "$BACKUP" "$DIR_DESTI"
        log "Backup restaurat a $DIR_DESTI"

}

    [[ "$ENTORN" == "$e" ]] && valid=true
done

DIR_DESTI="$DIR_DESTI_BASE/$ENTORN"

# Confirmació per a producció
if [[ "$ENTORN" == "prod" ]]; then
    echo -e "\e[31m⚠ ATENCIÓ: Estàs desplegant a PRODUCCIÓ!\e[0m"
    read -r -p "Escriu 'DESPLEGAR' per confirmar: " conf
    [[ "$conf" == "DESPLEGAR" ]] || { log "Desplegament cancel·lat"; exit 0; }
fi

log "Iniciant desplegament a $ENTORN..."

# Còpia de seguretat
if [[ -d "$DIR_DESTI" ]]; then
    BACKUP="$DIR_DESTI_backup_$(date +%Y%m%d_%H%M%S)"
    cp -r "$DIR_DESTI" "$BACKUP"
    log "Backup creat: $BACKUP"

# Desplegament
mkdir -p "$DIR_DESTI"
cp -r "$DIR_FONT"/. "$DIR_DESTI"/
log "Fitxers desplegats a $DIR_DESTI"

# Elimina backup si tot ha anat bé
